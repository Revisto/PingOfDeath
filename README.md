# Ping of Death
A simple and easy tool for skids to preform p.o.d. attacks

## How to Install
Run 'wget https://raw.githubusercontent.com/4k-developer/Ping_of_Death/master/pod && sudo chmod +x pod && sudo mv pod /usr/bin/' without the quotes

## How to Use
pod (ip address)

## How to Kill
Use 'pkill ping'

## What you Need
1) Ping 
2) A terminal
3) Target
4) Many attackers
